Overview of Dashboard

This project leverages Django's ORM to create a database from given JSON data and develops a dynamic data visualization dashboard. Utilizing JavaScript and Bootstrap for the front-end, the dashboard incorporates charts-sparkline.js for creative and interactive visualizations. Key variables such as Intensity, Likelihood, Relevance, Year, Country, Topics, and Region are dynamically visualized. The dashboard features user-friendly filters for End Year, Topics, Sector, Region, and Source, providing an intuitive and insightful exploration of the dataset.
